## 0.1 画布与背景

- 尺寸：**16:9（1920×1080）**
- 背景：**纯白 #FFFFFF** 为主 · 可叠加极淡浅青灰 **#F7FAF9** 局部渐变
- 背景纹理：右上角可保留极淡科技流线纹理（透明度 5-8% · 浅青灰 #E8F0EE）· 不抢眼
- 禁止：深色满铺背景 / 大面积非白底色（封面页除外）

| 用途 | 颜色 | 使用场景 |

## 0.4 品牌固定元素（每页必出现）

- 左下角：**01.AI 零一万物** Logo（小尺寸水印 · 深青绿色）
- 右上角（可选）：零一万物 01.AI 品牌标识（深青绿小字）
- 页码（可选）：右下角 12px #666666 · 两位数格式（如 07）
- 资料来源（可选）：左下角 Logo 上方 12px #666666

## 0.5 视觉质感语言

- **玻璃拟态 + 半透明亚克力质感**（克制 · 不要太"塑料"）
- **1px 细描边**（#D4E8E3 浅青灰 或 #5FD4C0 亮青绿）
- **轻青绿阴影**（4-8% 透明度 · 柔和投影）
- **图标统一风格**：半 3D 等距 + 青绿发光描边 + 透明玻璃底座
- **禁止**：卡通感 / 游戏感 / 霓虹蒸汽波 / 强对比阴影 / 大面积渐变

## 0.6 底部收束条（强烈推荐每页保留）

- 页面最底部 5% 高度
- 薄荷青绿半透明横条 或 深青绿实色条
- 居中放置 **16px 加粗** 收束句
- 用于本页的"一句话核心结论"

## 0.7 结构层完全开放

本规范不限制图形结构。卡片矩阵 / 流程图 / 时序图 / 关系网络 / 时间轴 / 四象限 / 桑基图 / 对比图均可。**哪个最能讲清逻辑就用哪个。**

## 0.9 图标系统（三档层级 · 智能空间适配）

| 层级 | 尺寸 | 使用位置 |
|---|---|---|
| L1 主图标 | 32-40px | 区域 / 模块的主标识 |
| L2 中图标 | 20-24px | 卡片 / 分层的标题前 |
| L3 小图标 | 14-18px | 具体功能项 / chip 标签前 |

**核心准则**：宁可不加 · 不可乱加。同层级要么都加 · 要么都不加。
